```
Step 1. 채널 필터 패널 (3행 계단식)
  채널(source): [전체] [Google] [네이버] [Meta] [카카오] [직접유입]
  매체(medium): [전체] [cpc] [display] ...   ← source 선택 시 노출
  캠페인:       [전체] [2025_0416] ...       ← medium 선택 시 노출

Step 2. 방문 페이지 목록
  URL | 세션수(정렬▼▲) | 평균 도달률(정렬▼▲) | 히트맵 버튼
  - maxH 320px 고정 + 내부 스크롤 (헤더 sticky)
  - 필터 조건에 매칭되는 raw_urls만 남긴 페이지 목록

Step 3. 스크롤 히트맵 (선택한 페이지 + 필터된 raw_urls 기준)
  - 기존 스크롤 히트맵과 동일 레이아웃
```
